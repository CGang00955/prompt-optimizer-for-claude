## ✨ 为什么选择Prompt Optimizer？

### 🎯 智能优化
自动检测提示词的四大要素（Persona、Task、Context、Format），智能补充缺失部分

### 🧠 元提示词技术
集成老金原创的元提示词强化技术，让提示词质量提升90%

### 📊 质量评分
实时评分系统（0-4分），帮助你不断改进提示词技巧

### 🔍 任务识别
智能识别任务类型，自动匹配最佳的角色和策略

## 🚀 快速开始

### 仅需一行命令

```bash
npx prompt-optimizer-for-claude
```

### 平台支持

| 平台 | 安装命令 |
|------|---------|
| **Windows** | `iwr -useb https://raw.githubusercontent.com/CGang00955/prompt-optimizer-for-claude/main/install.ps1 \| iex` |
| **Mac/Linux** | `curl -sSL https://raw.githubusercontent.com/CGang00955/prompt-optimizer-for-claude/main/install.sh \| bash` |

## 💡 使用示例

安装后，在Claude Code中：

### 输入：
```
帮我写个推广文案
```

### 自动优化为：
```
你是一位资深的营销策划师，擅长创意文案和品牌推广。

【核心任务】
请基于以下要求创作推广文案：
1. 目标受众是25-35岁的年轻消费者
2. 突出产品的核心卖点和独特优势
3. 使用吸引人的标题和强烈的行动号召
4. 保持文案简洁有力，控制在100字以内
5. 包含3个不同的版本供A/B测试

【上下文信息】
基于现代营销理论和消费心理学原理，结合社交媒体传播特性。

【输出格式】
- 版本一：突出功能特点
- 版本二：情感共鸣路线
- 版本三：限时促销导向
```
